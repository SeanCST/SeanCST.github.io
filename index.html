说明

本文档内的命令及说明是学习 廖雪峰老师的 Git 课程 时所做总结，以方便忘记命令时查看。

一、创建版本库

  • git init ： 把当前目录变成Git可以管理的仓库
  • git add <file>： 把文件添加到仓库 （把要提交的所有修改放到暂存区（Stage））
  • git commit -m <message>： 把文件提交到仓库，-m 后面输入的是本次提交的说明（一次性把暂存区的所有修改提交到分支）
