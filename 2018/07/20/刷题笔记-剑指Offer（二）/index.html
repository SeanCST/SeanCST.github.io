题目 - 数组中只出现一次的数字

题目描述

一个整型数组里除了两个数字之外，其他的数字都出现了偶数次。请写程序找出这两个只出现一次的数字。

解决方法

本题最直接的做法就是遍历数组，统计数组中每个数字出现的次数，可以由此找出只出现一次的两个数字。使用 map 便可很方便的统计出这一结果。

以上的方法比较通用，可以知道具体的每一个数字都出现了几次。但是，本题的特殊在于，并不需要知道出现偶数次的数字具体出现了几次，以上的统计之中就有很多内容显的多余，我们只需要知道此数字出现了偶数次，便有另外一个巧妙的方法：异或。

两个相同的数字异或的结果为 0。这一性质十分贴切于本题的情景，将所有出现了偶数次的数字一起异或，结果还是 0。故将数组内的所有数字异或后，结果与两个只出现一次的数字异或的结果相同，因其他数字异或的结果为 0。那么数组内数字全部异或之后的结果也只是我们需要找到的两个数字的异或结果，还是没有得到这两个数字的具体值。方法就是对异或的结果进行位处理，可以找到从最右边开始第一位不为 0 的位，此位不为 0 代表要求的两个数此位不相等，可以由此将两个数区分在两组之中，一个数组中的数该位为 0，另一个数组中的数该位不为 0。再将这两个数组中的数再分别全部异或，便自然地能得到要求的两个只出现一次的数。
